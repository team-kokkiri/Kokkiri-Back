# Kokkiri-Back

Kokkiri-Back is a Spring Boot project designed to provide a backend API service.

## Requirements

- **Java**: 17 or higher
- **Gradle**: 8.14 (wrapper included)

## Build and Run

```bash
# Build the project
./gradlew build

# Run the application
./gradlew bootRun
```

Alternatively, you can run the packaged JAR after building:

```bash
java -jar build/libs/*.jar
```
